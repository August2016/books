# Channel

Java Channel有如下几个特点:

* 一个channel支持同时读写
* channel支持异步读写
* channel总是同buffer配合进行读写

![](/assets/channel-and-buffer.png)

## Channel的实现

Java NIO中比较重要的几种channe实现如下:

* FileChannel : 文件读写
* DatagramChannel: 网络UDP读写
* SocketChannel: 网络TCP读写
* ServerSocketChannel: 作为server监听TCP连结请求
